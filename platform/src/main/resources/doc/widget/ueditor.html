描述 超级富文本编辑框(UEditor)
固定参数 API
序号 类型 描述
1 可选 宽度.
示例1(宽度800像素):ueditor[800px]
示例2(宽度占外部区域90%):ueditor[90%]
2 可选 高度.
示例1(宽度600像素,高度400像素):ueditor[600px;400px]
示例2(宽度使用系统默认值,高度200像素):ueditor[null;200px]
3 可选 工具栏模式.
normal(默认值):普通工具栏
dev:开发工具栏
mini:最小工具栏
动态参数 API
名称 类型 描述
(无) 此控件不支持动态入参,不支持表单校验(包括"是否为空"校验)
