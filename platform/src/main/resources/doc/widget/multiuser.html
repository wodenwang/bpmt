描述 用户选择(多选)
固定参数 API
序号 类型 描述
1 boolean 可选,是否在展示账号.默认false.
动态参数 API
名称 类型 描述
val string 默认值.
group boolean 是否展示组织架构,默认true.
query boolean flag:boolean类型,是否展示查询框,默认true.
mygroup boolean 是否展示"我的组织"快捷方式.默认值:true.
myself boolean 是否展示"我自己"快捷方式.默认值:true.
root string 组织架构树根节点.(在这个组织范围内查找用户)
black string/string[] 黑名单.(查询结果过滤)
white string/string[] 白名单.(查询结果仅包含)
