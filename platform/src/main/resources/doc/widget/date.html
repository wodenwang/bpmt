描述 时间选择框
固定参数 API
序号 类型 描述
1 可选 可选值:
date:默认值,格式如yyyy-MM-dd
datetime:格式如yyyy-MM-dd HH:mm:ss
time:格式如HH:mm:ss
yearmonth:格式如yyyyMM
2 可选 宽度.
示例1(宽度100像素):date[date;100px]
示例2(宽度占外部区域90%):date[datetime;90%]
3 可选 默认时间偏移量,遵循dateFormat规范.
示例1(30年以前,常用于生日):date[date;null;-30y]
示例2(一个月又10天):date[date;null;+1m 10d]
动态参数 API
名称 类型 描述
val string 默认值.now表示当前时间,其余情况下传入已格式化的时间字符.

enabled(flag) flag:true:可用,默认值;false:不可用. 将控件设置为可用/不可用(disabled)状态.

调用示例:
