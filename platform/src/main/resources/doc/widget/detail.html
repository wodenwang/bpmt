固定参数 API
序号 类型 描述
1 必填 自定义控件配置KEY.
动态参数 API
名称 类型 描述
orderBy string 排序SQL片段.
pageLimit number 每页记录数.
add boolean 控件是否允许增加数据.
delete boolean 控件是否允许删除数据.
? ? 任意自定义参数均可以在控件中的处理器通过cm.params()函数调用.
