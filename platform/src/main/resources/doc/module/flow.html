动态入参API(JSON格式)
名称 类型 描述
panel string 目标控制台.task:任务控制台,pd:流程控制台.默认值:task.使用pd(流程控制台时),可配合pdKey一起使用.
例:
只展示hetong和leave两种流程:{panel:'pd',pdKey:['hetong','leave']}
remove boolean 删除草稿箱订单,注:此项必须配合pdId和ordId使用.
例:
任务处理: {remove:true,ordId:'${vo.ORD_ID}',pdId:'${vo.PD_ID}'}
detail boolean 是否直接跳转到订单明细页,注:此项必须配合pdId和ordId使用;也可配合ordFlag使用.
例:
订单查看: {detail:true,ordId:'${vo.ORD_ID}',pdId:'${vo.PD_ID}'}
form boolean 是否直接跳转到任务处理页,注:处理任务此项必须配合pdId和ordId使用,新建订单时此项必须配合pdKey使用.
例:
任务处理: {form:true,ordId:'${vo.ORD_ID}',pdId:'${vo.PD_ID}'}
订单新建: {form:true,pdKey:'flow'}
pdKey string 流程定义主键(即"流程设置"中流程的唯一主键).
quickMode boolean 快速模式,当true是采用弹框方式处理任务,默认false.
pdId string 流程定义ID.
ordId string 订单号.
ordFlag boolean 查看时是否使用"查看订单"界面,不设置此项默认为false.当detail为true时生效.
